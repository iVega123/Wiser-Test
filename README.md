# Wiser Test

Projeto feito com base nos requisitos apresentados no teste da Wiser!

## Começando
Apenas clonar ou baixar o repositório e descompactar!
### Pré-Requisitos

```
Node js e NPM
```

### Instalando
Você precisa seguir os seguintes passos!

```
npm install
```
Quando finalizado
```
npm start
```
